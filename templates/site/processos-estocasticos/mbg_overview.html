{% if request.session.language == 'en' %}1. Generalized Brownian Process Wiener{% else %}1. Processo Browniano Generalizado Wiener{% endif %}

{% if request.session.language == 'en' %}Concept:{% else %}Conceito:{% endif %} {% if request.session.language == 'en' %} The Wiener process, also known as standard Brownian motion ($W_t$), is a continuous-time stochastic process that models the random movement of particles. It is characterized by independent and normally distributed increments, with mean zero and variance equal to the time interval. In finance, it forms the basis for modeling the random component of asset prices, where the price changes are assumed to follow a random walk. A generalized Brownian process is often represented as $dX_t = \mu dt + \sigma dW_t$, where $\mu$ is the drift and $\sigma$ is the volatility. {% else %} O processo de Wiener, também conhecido como movimento Browniano padrão ($W_t$), é um processo estocástico de tempo contínuo que modela o movimento aleatório de partículas. É caracterizado por incrementos independentes e normalmente distribuídos, com média zero e variância igual ao intervalo de tempo. Em finanças, ele forma a base para modelar o componente aleatório dos preços dos ativos, onde as mudanças de preço são assumidas como seguindo um passeio aleatório. Um processo Browniano generalizado é frequentemente representado como $dX_t = \mu dt + \sigma dW_t$, onde $\mu$ é o drift e $\sigma$ é a volatilidade. {% endif %}

{% if request.session.language == 'en' %}2. Generalized Brownian Process Ito{% else %}2. Processo Browniano Generalizado Ito{% endif %}

{% if request.session.language == 'en' %}Concept:{% else %}Conceito:{% endif %} {% if request.session.language == 'en' %} Ito's Lemma is a fundamental result in stochastic calculus that allows us to find the differential of a function of a stochastic process. When dealing with functions of a Wiener process, standard calculus rules do not apply due to the non-differentiability of the Wiener process. Ito's Lemma extends the chain rule to stochastic processes, enabling the analysis of asset prices that follow Geometric Brownian Motion. For a function $f(X_t, t)$ where $dX_t = \mu dt + \sigma dW_t$, Ito's Lemma states $df = \left(\frac{\partial f}{\partial t} + \mu \frac{\partial f}{\partial X} + \frac{1}{2}\sigma^2 \frac{\partial^2 f}{\partial X^2}\right) dt + \sigma \frac{\partial f}{\partial X} dW_t$. This is crucial for deriving models like Black-Scholes. {% else %} O Lema de Ito é um resultado fundamental no cálculo estocástico que nos permite encontrar o diferencial de uma função de um processo estocástico. Ao lidar com funções de um processo de Wiener, as regras de cálculo padrão não se aplicam devido à não diferenciabilidade do processo de Wiener. O Lema de Ito estende a regra da cadeia para processos estocásticos, permitindo a análise de preços de ativos que seguem o Movimento Browniano Geométrico. Para uma função $f(X_t, t)$ onde $dX_t = \mu dt + \sigma dW_t$, o Lema de Ito afirma $df = \left(\frac{\partial f}{\partial t} + \mu \frac{\partial f}{\partial X} + \frac{1}{2}\sigma^2 \frac{\partial^2 f}{\partial X^2}\right) dt + \sigma \frac{\partial f}{\partial X} dW_t$. Isso é crucial para derivar modelos como Black-Scholes. {% endif %}
