{% if request.session.language == 'en' %}1. Random Walk - Discrete Time{% else %}1. Passeio Aleatório - Tempo Discreto{% endif %}

{% if request.session.language == 'en' %}Concept:{% else %}Conceito:{% endif %} {% if request.session.language == 'en' %} A discrete-time random walk describes a path consisting of a sequence of random steps on some mathematical space. At each step, the position of the "walker" changes by a random amount. This model is fundamental in various fields, including finance (modeling stock prices), physics (Brownian motion), and computer science, illustrating basic principles of randomness and probability. {% else %} Um passeio aleatório em tempo discreto descreve um caminho que consiste em uma sequência de passos aleatórios em algum espaço matemático. A cada passo, a posição do "caminhante" muda por uma quantidade aleatória. Este modelo é fundamental em várias áreas, incluindo finanças (modelagem de preços de ações), física (movimento Browniano) e ciência da computação, ilustrando princípios básicos de aleatoriedade e probabilidade. {% endif %}

{% if request.session.language == 'en' %}2. Random Walk - Normal View{% else %}2. Passeio Aleatório - Visão Normal{% endif %}

{% if request.session.language == 'en' %}Concept:{% else %}Conceito:{% endif %} {% if request.session.language == 'en' %} The "Normal View" of a random walk focuses on visualizing how the distribution of positions converges to a normal (Gaussian) distribution as the number of steps increases. This demonstrates the Central Limit Theorem in action, showing that the sum of many independent random variables (the individual steps) tends towards a normal distribution, regardless of the distribution of the individual steps themselves. This provides key insights into the statistical properties of random walks over longer periods. {% else %} A "Visão Normal" de um passeio aleatório foca em visualizar como a distribuição das posições converge para uma distribuição normal (Gaussiana) à medida que o número de passos aumenta. Isso demonstra o Teorema do Limite Central em ação, mostrando que a soma de muitas variáveis aleatórias independentes (os passos individuais) tende a uma distribuição normal, independentemente da distribuição dos passos individuais. Isso fornece insights importantes sobre as propriedades estatísticas dos passeios aleatórios em períodos mais longos. {% endif %}
