{% if request.session.language == 'en' %}Method Description{% else %}Descrição do Método{% endif %}

{% if request.session.language == 'en' %} The Black-Scholes-Merton model expands on the classic Black-Scholes by incorporating dividend yields. Developed by Robert Merton, this extension handles dividend-paying stocks by adjusting the underlying asset price to account for the expected dividends during the option's lifetime. {% else %} O modelo Black-Scholes-Merton expande o clássico Black-Scholes ao incorporar rendimentos de dividendos. Desenvolvido por Robert Merton, esta extensão trata de ações com pagamento de dividendos ajustando o preço do ativo subjacente para considerar os dividendos esperados durante a vida da opção. {% endif %}

{% if request.session.language == 'en' %} Key Parameters: Underlying asset price (S), Strike price (K), Time to expiration (T), Risk-free rate (r), Volatility (σ), and Dividend yield (q) {% else %} Parâmetros Principais: Preço do ativo subjacente (S), Preço de exercício (K), Tempo até o vencimento (T), Taxa livre de risco (r), Volatilidade (σ), e Rendimento de dividendos (q) {% endif %}

{% if request.session.language == 'en' %} This model is particularly useful for valuing options on dividend-paying stocks, indices, and other assets with continuous yield. {% else %} Este modelo é particularmente útil para avaliar opções sobre ações que pagam dividendos, índices e outros ativos com rendimento contínuo. {% endif %}
