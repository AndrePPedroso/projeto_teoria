{% if request.session.language == 'en' %}2.1 Strategy 1: Purchase of the Asset and a Put Option{% else %}2.1 Estratégia 1: Compra do Ativo e da Opção de Venda{% endif %}

{% if request.session.language == 'en' %}Objective:{% else %}Objetivo:{% endif %} {% if request.session.language == 'en' %} To simultaneously buy a put option on an asset and the asset itself. This is a classic protective strategy, where the put acts as an insurance policy against a fall in the asset's price. {% else %} Comprar simultaneamente uma opção de venda sobre um ativo e o próprio ativo. Trata-se de uma estratégia clássica de proteção (hedge), onde a put funciona como um seguro contra a queda no preço do ativo. {% endif %}

{% if request.session.language == 'en' %}2.2 Strategy 2: Collar Strategy{% else %}2.2 Estratégia 2: Estratégia Collar{% endif %}

{% if request.session.language == 'en' %}Objective:{% else %}Objetivo:{% endif %} {% if request.session.language == 'en' %} To simultaneously buy a put option and the asset itself, in addition to selling a call option. In certain conditions, this strategy locks in a fixed result, demonstrating the put-call parity relationship (Correia Neto, 2011). {% else %} Comprar simultaneamente uma opção de venda sobre um ativo e o próprio ativo, além de uma venda de uma opção de compra. Sob certas condições, essa estratégia trava um resultado fixo, o que evidencia a relação de paridade entre opções de compra e venda (Correia Neto, 2011). {% endif %}
